# Crime Incident Report System

A web-based, database-driven application designed to digitally record, manage, and analyze crime-related incidents. The system allows users to submit crime reports, while police authorities and administrators manage reports, stations, users, and analytics through role-based dashboards.

This project was developed as part of **CSE370: Database Systems**.
